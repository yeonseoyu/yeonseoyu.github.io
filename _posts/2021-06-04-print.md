## 1. 정수의 출력  
---
정수를 출력할 때는 %d 서식지정자를 사용합니다.  
~~~c
#include <stdio.h>

int main()
/*
{
 printf("a=%d\n", 123);
 printf("b=%4.1f\n", 6.42);
 printf("a의 값은 %d이다.\n", 456);
 printf("b의 값은 %4.1f이다.\n", 642);
 return 0;
}
~~~

[출력결과]
a=123  
b=6.3  
a의 값은 456이다  
b의 값은 6.4이다
